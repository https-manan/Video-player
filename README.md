# Video Player

A simple and responsive video player built with HTML, CSS, and JavaScript.

## Features
- Play, pause, and seek videos  
- Adjustable volume and playback speed  
- Fullscreen toggle  
- Custom controls with clean UI  

## Technologies Used
- HTML5 Video API  
- CSS3  
- JavaScript (Vanilla)
